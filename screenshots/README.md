## How to Capture Screenshots

### Using Xcode Simulator:

1. Build and run the app in the iOS Simulator
2. Navigate to the screen you want to capture
3. Press `Cmd + S` to save a screenshot, or
4. Use the simulator menu: File → Save Screenshot

### Using iOS Simulator with Different Devices:

1. In Xcode, select different simulator devices (iPhone 15, iPhone SE, etc.)
2. Run the app and capture screenshots for each device size
3. This helps show how the app looks on different screen sizes

### Screenshot Specifications:

- **Format**: PNG (preferred) or JPG
- **Resolution**: Native device resolution (recommended) or scaled
- **Orientation**: Portrait (default)
- **File size**: Keep under 2MB each for GitHub

## Tips for Great Screenshots:

- Use realistic sample data
- Show the app in both light and dark mode if applicable
- Remove any sensitive/personal data
- Use the latest iOS version for screenshots
- Consider adding device frames using tools like [Frames](https://frames.app/) or [MockUPhone](https://mockuphone.com/)
